# My_Routine
Arghya's Weekly Routine Planner
A lightweight, fully offline weekly schedule planner for university students — built with pure HTML, CSS, and JavaScript.
What it does
The planner shows your full 6-day week (Sunday to Friday) on a precise time grid from 8:00 AM to 10:00 PM. Classes are positioned by exact minute so broken times like 10:20 AM sit at their real pixel position. Each course gets a distinct Material Design color so you can read the schedule at a glance. Free time gaps between classes are calculated automatically and shown with their total duration per day.
Features
Edit Mode lets you do everything from inside the browser. Click any class to edit its name, day, time, room, type, and color. Hover a block to delete it. Add new entries for classes, gym sessions, meals, or anything else. Label any free gap with a custom activity. All changes save instantly to localStorage with no save button needed. Dark mode and light mode are both available and your preference is remembered on next open.
Tech
HTML · CSS · Vanilla JavaScript · localStorage · Google Fonts (DM Sans + Space Mono)
How to use
Download the HTML file. Open it in any browser. That is it. No server, no npm, no internet required after download. Your schedule and all edits persist automatically in your browser's local storage.
